My Projects

This is a list of most of the projects that I have been working on. Feel free to click the links and to check them out. There are some fun Web Apps, and a few websites, some with databases or APIs incorporations.

  • Tennis4Life This is a Frontend and Backend project for tennis fans. It allows users to sign up and post their favorite tennis clubs. The website was build using MongoDB as a database. The website is fully functional, you can add tennis clubs, comments, edit them. There is also a google map API on the tennis clubs page. Tennis4Life was created using Node.js with AWS Cloud 9 and deployed on Heroku.
  • YelpCamp: This is another Frontend and Backend project to list campgrounds. Users can sign up and login to post their favorite campgrounds with pictures and comments. The website was build using MongoDB as a database. The website is fully functional and comes with a google map API to locate the campgrounds. YelpCamp was created using Node.js with goormIDE and deployed on Heroku
  • TravelWell Blog: This is a blog to post travel destinations. There is no database so no sign up is required. TravelWell was created using Node.js with goormIDE and deployed on Heroku
  • Movie App: This is a Web App to find some movies related to a specifc theme or keyword. I uesd the OMDb API for this app. Movie App was created using Node.js with goormIDE and deployed on Heroku.
  • To Do List: This is nice Web App to create a To Do List. You can add items and delete items on the list. To-Do-List was designed using HTML, CSS, JavaScript and jQuery.
  • Color Game: This is a Web App game where you have to try to guess the RGB color. It comes with some functions that allows players to reset the game or chose a different level. Color Game was designed using HTML, CSS, JavaScript.
  • Scores: This is a basic score keeper page that can be used for playing games or sports. Score Keeper was designed using HTML, CSS, JavaScript.
  • Fitness Home Workouts: This is a a website that I am currently deisigning and maintaining in my current job. It allows my students to have access to a variety or resources and fitness programs. This website is designed with HTML, CSS, Bootstrap, JavaScirpt and jQuery.
  • Gallery: This is a webpage or template that can be used to display an image gallery. This page was designed with HTML and CSS.
  • Foodar: This was my first real project and I am in the process of upgrading it. It is a food delivery assignment from the company Revature.
